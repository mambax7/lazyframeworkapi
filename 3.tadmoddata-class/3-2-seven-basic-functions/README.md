# 3-2 Seven basic functions

### 3-2 Seven basic functions

Common CRUD functions

1. `create()`: Add and create forms
2. `edit($id)`: Modify the form //$id is Primary key
3. `show($id)`: Display single content
4. `update($id)`: Update
5. `store()`: Save to add
6. `destroy($id)`: Delete data
7. `index()`: List all
